How to deploy your github page with python pelican?

Post in Tue 05 May 2020 |Tags

Steps:

  1. pip install pelican Markdown
  2. mkdir YOUR_PAGE_DIR & cd YOUR_PAGE_DIR
  3. pelican-quickstart
  4. write a markdown page in ./content, e.g.:
Title: My First Review
Date: 2010-12-03 10:20
Category: Review

Following is a review of my favorite mechanical keyboard.
  1. pelican content
  2. cd output
  3. python -m pelican.server
  4. preview the page at localhost:8000
  5. pull your github page repository (YOUR_GITHUB_NAME.github.io), replace files inside with those in ./output
  6. push and view your page.

A simpler way

  1. Write your blog within ./content, naming the .md file as "yyyy_mm_dd.md".
  2. Must included are Title, Date, and Category.
  3. Key in pelican content command in terminal.
  4. Copy & paste all from ./output to your github page folder.
