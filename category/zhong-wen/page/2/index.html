更新到指定版本

npm -g install npm@2.9.1

因为国内的网络环境原因，在下载npm包的时候经常会遇到无法正常下载的情况。国内淘宝推出了 npm 镜像，这是一个完整的 npmjs.org 镜像，同步频率为15分钟一次，保证与官方服务同步。推荐使用该镜像。使用方法如下：

npm install -g cnpm --registry=http://registry.npm.taobao.org
